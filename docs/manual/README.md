## 更新ルール

リリース時は、アプリの変更とあわせてこのマニュアルも更新してください。

1. アプリを最新状態でビルドします。
2. 実画面のスクリーンショットを撮り直します。
3. 認証情報、メールアドレス、社員番号、実業務コメントなどが写っていないか確認し、必要に応じてマスクします。
4. タブ構成、入力項目、申請・承認の動作、注意事項を更新します。
5. 配布用ZIPへ `docs/manual` を同梱します。

スクリーンショット更新の補助スクリプト:

```powershell
node scripts/capture-manual-screenshots.mjs
```

補助スクリプトは主要タブの画面を `docs/manual/images` に保存し、設定画面などの機密値をマスクします。
