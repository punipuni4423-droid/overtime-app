freee申請ツール スタートアップ手順書

対象バージョン: 1.0.13

1. 利用前に準備するもの

必要なもの:

  • Windows 10 / Windows 11 のPC
  • インターネット接続
  • freee人事労務にログインできるアカウント
  • freee APIのOAuthアプリ情報
  • Client ID
  • Client Secret
  • Microsoft Edge または Google Chrome

通常利用では、Node.jsや開発ツールのインストールは不要です。

Web自動操作を使う機能では、PCにMicrosoft EdgeまたはGoogle Chromeが必要です。EdgeはWindows標準で入っていることが多いため、通常は追加インストール不要です。

2. インストール方法

インストーラー版

  1. GitHub Releaseから overtime-app_Setup_1.0.13.exe をダウンロードします。
  2. ダウンロードしたexeを実行します。
  3. 画面の案内に従ってインストールします。
  4. インストール後、アプリを起動します。

配布用ZIP版

  1. 配布用ZIPをダウンロードします。
  2. 任意のフォルダに展開します。
  3. 展開したフォルダ内の 残業申請ツール.exe を実行します。

ZIP版はインストールせずに実行できます。社内共有フォルダなどで配布する場合に利用してください。

3. 初回設定

アプリ起動後、右上の歯車アイコンから設定画面を開きます。

OAuth2認証情報

  1. Client ID を入力します。
  2. Client Secret を入力します。
  3. 設定を保存 を押します。
  4. OAuth認証ボタンを押し、ブラウザでfreeeへログインして認可します。
  5. 認証後、Company ID / User ID / Employee ID が自動取得されます。

デフォルト部門

部門指定が必要な申請経路を使う場合は、デフォルト部門 を選択してください。

部門・役職ベースの承認経路では、API申請が制限される場合があります。その場合、アプリはWeb版freeeの自動操作に切り替えて申請します。

Web版自動操作用ログイン情報

一部の申請や承認操作では、freeeのWeb画面を自動操作します。

  1. freeeログイン用のメールアドレスを入力します。
  2. パスワードを入力します。
  3. ログイン確認 を押します。
  4. ログイン確認済み と表示されることを確認します。

ログイン確認が未完了の場合、Web操作が必要な申請・承認ボタンは押せません。

4. 起動確認

  1. アプリ上部の 残業申請 タブを開きます。
  2. 申請経路が表示されることを確認します。
  3. 申請・承認 タブを開き、申請一覧を取得できることを確認します。
  4. Manager権限がある場合は、残業状況 タブで対象月を選択して取得できることを確認します。

5. 自動承認を使う場合

自動承認はWindowsのタスクスケジューラーを使用します。

確認すること:

  • PCが起動していること
  • Windowsタスクスケジューラーが利用できること
  • アプリの設定でWebログイン確認が完了していること
  • 申請・承認 タブで自動承認の時間と経路が設定されていること

自動承認設定を保存すると、タスクスケジューラーの存在と有効状態を確認します。タスクがない場合は作成されます。

6. よくあるトラブル

申請経路が表示されない

  • OAuth認証が完了しているか確認してください。
  • Client ID / Client Secret が正しいか確認してください。
  • freeeアカウントに対象会社の権限があるか確認してください。

Web経由申請に切り替わる

部門・役職ベースの承認経路など、freee APIで申請できない条件の場合に発生します。

  • 設定画面でメールアドレス・パスワードを入力してください。
  • ログイン確認 を完了してください。
  • EdgeまたはChromeが利用できる状態か確認してください。

休日出勤で平日が選べない

仕様です。休日出勤申請は土日祝日のみ選択できます。

有給申請で土日祝日が除外される

仕様です。有給申請は営業日を対象とするため、土日祝日は除外されます。

残業状況が取得できない

  • Manager権限があるか確認してください。
  • freee側で他メンバーの月次勤怠集計を閲覧できる権限が必要です。
  • OAuth認証が有効か確認してください。

7. 更新方法

インストーラー版:

  1. 新しい overtime-app_Setup_X.Y.Z.exe をダウンロードします。
  2. 実行して上書きインストールします。

ZIP版:

  1. 新しい配布用ZIPをダウンロードします。
  2. 古いフォルダとは別の場所へ展開します。
  3. 必要に応じてショートカットを差し替えます。

設定情報はユーザー環境に保存されるため、通常は再入力不要です。
